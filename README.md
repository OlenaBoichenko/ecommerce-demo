# ShopDemo - E-Commerce Demo Platform

A full-stack e-commerce solution built with Next.js, featuring real-time inventory management, secure payment processing with Stripe, and an intuitive admin dashboard.

## Features

- **Product Catalog** with advanced filtering by category, price range, and search
- **Product Detail Pages** with quantity selection and stock tracking
- **Shopping Cart** with real-time updates and persistence
- **Wishlist Functionality** to save favorite products
- **Checkout Flow** with Stripe payment integration (demo mode)
- **Order Tracking System** to monitor order status
- **Admin Dashboard** with inventory and order management
- **Dark/Light Theme Toggle** for better user experience
- **Responsive Design** that works on all devices

## Tech Stack

- **Frontend**: Next.js, React, TypeScript
- **Styling**: Tailwind 
- **State Management**: Zustand 
- **Database**: MySQL
- **Payment Processing**: Stripe
